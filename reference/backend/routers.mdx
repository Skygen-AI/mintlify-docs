## Overview

All routes are mounted on a single FastAPI application at `app/main.py` using `app.include_router()`. The Socket.IO server wraps it via `get_socket_app(app)` → `socket_app` (the actual uvicorn target).

**Auth:** Unless stated otherwise, endpoints require `Authorization: Bearer <jwt>` via `get_current_active_user`. Admin-only endpoints additionally require `role=admin` or admin-panel JWT.

### /agent — Device Control & Task Management

**Auth:** All device/screenshot endpoints require agent token (`verify_agent_auth`). Task cancel/status endpoints use standard user JWT (`get_current_active_user`).

Source: `app/routers/agent/device_control.py`

| Method | Path | Auth | Description |
|---|---|---|---|
| `POST` | `/agent/device/execute` | Agent token | Execute a device action. Body: `{device_id, action_type, parameters?, timeout, action_id?}`. Routes to WebSocket for real devices or E2B SDK for sandbox devices. Returns `DeviceActionResponse {success, action_id, task_id?, result?, error?, execution_time?, status}`. |
| `GET` | `/agent/device/action/{action_id}/result` | Agent token | Poll for action result. Returns same `DeviceActionResponse`. Status values: `completed` / `failed` / `pending` / `timeout` / `not_found`. Results cached in Redis with TTL. |
| `GET` | `/agent/device/{device_id}/status` | Agent token | Get device connection status. Returns `{device_id, is_connected, connection_info}`. Works for real and E2B devices. |
| `POST` | `/agent/upload-screenshot/presigned` | Agent token | Get S3 presigned upload URL for a screenshot. Body: `{session_id, task_id?}`. Returns `{upload_url, fields, screenshot_id, expires_at, method}`. |
| `POST` | `/agent/upload-screenshot` | Agent token | Legacy direct screenshot upload (base64, max 10 MB). Body: `{session_id, step_number, screenshot_base64}`. Returns `{screenshot_url, status: "uploading"}`. |
| `POST` | `/agent/task/cancel` | User JWT | Cancel a running agent task. Body: `{session_id, device_id?, reason?}`. Sets Redis cancel flag, wakes any hibernated agents, saves cancel message to DB. Returns `TaskCancellationResponse {success, session_id, message, is_active, device_notified}`. |
| `GET` | `/agent/task/{session_id}/status` | User JWT | Check task status for a session. Returns `{session_id, is_active, is_cancelled, status}` where `status` is one of `active` / `cancelling` / `cancelled` / `completed_or_not_found`. |
