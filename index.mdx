## What is Skygen?

Skygen is a multi-platform AI agent automation platform that lets you build, connect, and run intelligent agents across real computers, cloud desktops, and software environments. Whether you need an agent to browse the web, fill out forms, write code, send emails, or query APIs — Skygen provides the infrastructure to make it happen reliably at scale.

Agents communicate with users in real time over WebSockets, execute multi-step plans across GUI and API surfaces, and can be triggered manually, via API, or on a cron schedule. Every action is logged, every cost is tracked, and every integration is MCP-native — so you can connect Slack, Gmail, GitHub, and dozens of other tools without writing custom glue code.

## Architecture Overview

```
┌─────────────────────────────────────────────────────────────────┐
│                         Skygen Platform                         │
│                                                                 │
│   WebApp / Mobile App / API client                              │
│          │                                                      │
│          ▼  REST + Socket.IO                                    │
│   ┌─────────────────┐    ┌──────────────┐   ┌──────────────┐   │
│   │  Backend API    │    │  RabbitMQ    │   │  Redis       │   │
│   │  (FastAPI)      │───▶│  (task queue)│   │  (pub/sub,   │   │
│   │  Port 8000      │    └──────┬───────┘   │   sessions)  │   │
│   └────────┬────────┘           │           └──────────────┘   │
│            │ JWT auth            ▼                               │
│            │           ┌─────────────────┐                      │
│            │           │  Agent Service  │                      │
│            │           │  (Python 3.12)  │                      │
│            │           │  Port 8080      │                      │
│            │           └────────┬────────┘                      │
│            │                    │                               │
│            │      ┌─────────────┼─────────────┐                 │
│            │      ▼             ▼             ▼                 │
│            │  Cloud PC      User Device   Software              │
│            │  (E2B VM)      (Mac/Win)     (API/Code)            │
│            │                                                    │
│            ▼                                                    │
│   ┌─────────────────┐   ┌──────────┐   ┌──────────┐            │
│   │  PostgreSQL      │   │  S3/MinIO│   │  MCP     │            │
│   │  (data store)    │   │  (files) │   │  tools   │            │
│   └─────────────────┘   └──────────┘   └──────────┘            │
└─────────────────────────────────────────────────────────────────┘
```

**Key components:**

- **Backend API** — FastAPI REST server + Socket.IO real-time layer. Handles auth, sessions, billing, device management, and message routing.
- **Agent Service** — Python 3.12+ multi-agent framework. Receives tasks from RabbitMQ, routes them to the right compute mode, runs the LLM loop, and streams results back over Socket.IO.
- **Socket.IO namespaces** — `/user` for frontend notifications, `/device` for device commands, `/agent` for agent subscriptions.
- **Storage** — PostgreSQL (primary data), Redis (session state + pub/sub for horizontal scaling), S3/MinIO (file attachments), RabbitMQ (async task dispatch).
- **MCP integrations** — Composio-backed connectors for Slack, Gmail, GitHub, Google Calendar, Google Drive, and more.

## Pricing at a Glance

Skygen uses a token-based billing model. Your subscription includes a monthly token allowance, and you can enable overage billing to keep agents running beyond the included amount.

| Plan | Tokens/month | Overage |
|------|-------------|---------|
| Standard ($14.99/mo) | 2,000 tokens | $0.0075 / token |
| Free trial | Limited | — |

Tokens represent a normalized unit of LLM API cost with a 1.5× platform margin. Cloud PC usage is billed separately at $0.1656/hr (2 vCPU, 4 GiB RAM) with a 2.5× markup.

See the [Pricing guide](/guides/pricing) for full details.
