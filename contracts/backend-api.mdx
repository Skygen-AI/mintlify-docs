## Auth conventions

| Method | How it works |
|--------|-------------|
| **Bearer JWT** | `Authorization: Bearer <token>` header **or** `skygen_session` cookie |
| **Agent Token** | `Authorization: Bearer <AGENT_API_TOKEN>` or `X-Agent-Token: <token>` header |
| **Internal Token** | `X-Internal-Token` header — blocked by `InternalRouteGuard` for external (ALB) traffic |
| **Admin JWT** | Bearer JWT where `user.is_admin == True` |
| **HTTP Basic** | `admin` / `skygen_docs_pass!2026*` — only for `/docs` and `/openapi.json` |

## Auth — `/auth`

### POST /auth/register
**Auth:** none
**Request body:**
- `email`: str (required)
- `password`: str (required)
- `display_name`: str (optional)
- `invite_code`: str (optional, required when `REGISTRATION_MODE=invite_only`)
- `meta`: dict (optional — UTM/GA params)

**Response 201:**
```json
{"user": <UserRead>, "access_token": "jwt", "token_type": "bearer"}
```
Sets `skygen_session` cookie.
**Response 400:** email taken, invalid invite code, expired invite
**Side effects:** inserts `users`, optionally updates `invite_codes` (marks used), inserts `user_email_segments`, fires `registration_completed` analytics event, sends verification email via SES

### GET /auth/verify?token=
**Auth:** none (link from verification email)
**Response 200:** `{"msg": "Email successfully verified"}`
**Response 400:** invalid/expired token
**Side effects:** sets `users.is_email_verified=True`, sends welcome email

### POST /auth/token (login)
**Auth:** none
**Request:** `application/x-www-form-urlencoded` — `username`, `password`
**Response 200:**
```json
{"access_token": "jwt", "token_type": "bearer"}
```
Sets `skygen_session` cookie.
**Response 401:** invalid credentials
**Rate limit:** login_rate_limit

### POST /auth/logout
**Auth:** none (cookie present or not)
**Response 204:** empty
**Side effects:** clears `skygen_session` and `skygen_logged_in` cookies

### POST /auth/forgot-password
**Auth:** none
**Request body:** `{"email": "str"}`
**Response 202:** `{"msg": "..."}` (always 202, no enumeration)
**Rate limit:** forgot_password_rate_limit
**Side effects:** sends password reset email if user exists

### POST /auth/reset-password
**Auth:** none
**Request body:** `{"token": "str", "new_password": "str"}`
**Response 200:** `{"msg": "Password successfully reset"}`
**Response 400:** invalid/expired token
**Side effects:** updates `users.password_hash` and `users.password_changed_at`

### POST /auth/upgrade-session
**Auth:** `Authorization: Bearer <legacy-token>`
**Response 204:** Sets `skygen_session` cookie
**Response 410:** `AUTH_LEGACY_UPGRADE_DISABLED=True`
**Side effects:** pins localStorage JWT into cookie

### GET /auth/google
**Auth:** none
**Response 302:** redirect to Google OAuth consent

### GET /auth/google/callback?code=&state=
**Auth:** none (OAuth callback)
**Response 302:** redirect to `/app/home` (success) or `/app/login?google_error=` (failure)
**Side effects:** creates/links Google user in `users`, sends `registration_completed` event, sets `skygen_session` cookie

### POST /auth/google-mobile/token
**Auth:** none
**Request body:** `{"id_token": "google_id_token_str"}`
**Response 200:** `{"access_token": "jwt", "token_type": "bearer"}`
**Response 401:** invalid Google token

### PATCH /auth/update-registration-meta
**Auth:** Bearer JWT
**Request body:** `{"meta": {"ga_client_id": ..., "utm_source": ...}}`
**Response 200:** `{"status": "ok"}`
**Side effects:** merges into `users.registration_meta`, re-fires `registration_completed` analytics

### POST /auth/setup-2fa
**Auth:** Bearer JWT
**Response 200:** `{"secret": "", "uri": "", "is_new": false, "message": "..."}` *(2FA disabled stub)*

### POST /auth/verify-2fa
**Auth:** Bearer JWT
**Request body:** `{"otp_code": "str"}`
**Response 200:** `{"success": false, ...}` *(2FA disabled stub)*

### GET /auth/2fa-status
**Auth:** Bearer JWT
**Response 200:** `{"is_enabled": false, "is_session_valid": true, ...}` *(2FA disabled stub)*

## Sessions — `/sessions`

### POST /sessions
**Auth:** Bearer JWT
**Request body:** `ChatSessionCreate`
- `title`: str (optional)
- `device_id`: UUID (optional)
- `meta_data`: dict (optional)
- `emplogent_id`: UUID (optional)

**Query param:** `reuse_existing: bool = False`
**Response 201:** `ChatSessionRead`
**Response 403:** device belongs to another user
**Response 404:** device not found
**Side effects:** inserts `chat_sessions`, emits `session_created` to `/user` Socket.IO namespace

### GET /sessions
**Auth:** Bearer JWT
**Query:** `limit` (1-100, default 20), `offset`
**Response 200:** `{"items": List[ChatSessionRead], "total": int, "limit": int, "offset": int}`

### GET /sessions/{session_id}
**Auth:** Bearer JWT
**Response 200:** `ChatSessionRead`
**Response 403:** session belongs to another user
**Response 404:** session not found

### PATCH /sessions/{session_id}
**Auth:** Bearer JWT
**Request body:** `ChatSessionUpdate` (partial — `title`, `meta_data`)
**Response 200:** `ChatSessionRead`
**Side effects:** updates `chat_sessions`, emits `session_updated` to `/user`

### DELETE /sessions/{session_id}
**Auth:** Bearer JWT
**Response 204:**
**Side effects:** sets `chat_sessions.deleted_at` (soft delete), releases Redis device/session locks, emits `session_deleted`

### DELETE /sessions/{session_id}/permanent
**Auth:** Bearer JWT
**Response 204:**
**Side effects:** hard deletes session and all messages, releases locks, emits `session_deleted`

### POST /sessions/{session_id}/restore
**Auth:** Bearer JWT
**Response 200:** `ChatSessionRead`
**Side effects:** clears `chat_sessions.deleted_at`, emits `session_restored`

### GET /sessions/{session_id}/plan
**Auth:** Bearer JWT
**Response 200:** `{"plan": <plan_payload>|null}`
**Used by:** WebApp session reload (restore agent plan steps)

### GET /sessions/{session_id}/summary
**Auth:** Bearer JWT
**Query:** `max_messages` (1-50, default 10)
**Response 200:** `{"session_id", "total_messages", "summary_messages", "messages": [...]}`

## Messages — `/sessions/{session_id}/messages`

### POST /sessions/{session_id}/messages
**Auth:** Bearer JWT
**Request body:**
- `role`: str (required — `"user"`)
- `content`: str (required)
- `meta_data`: dict (optional)
- `task_id`: UUID (optional)
- `agent_mode`: str (default `"auto"`)
- `client_version`: str (optional)
- `device_id`: UUID (optional)

### DELETE /sessions/{session_id}/messages/{message_id}

**Response 204:**
**Side effects:** deletes `chat_messages`, emits `message_deleted` to `/user`

### POST /sessions/{session_id}/messages/{message_id}/rate
**Auth:** Bearer JWT
**Request body:** `{"rating": int (1-5), "comment": str|null}`
**Response 200:** `{"success": true, "rating": int, "comment": str|null}`
**Side effects:** upserts `message_ratings`, fires Telegram alert if rating ≤ 2

### POST /audio/transcribe
**Auth:** Bearer JWT
**Request:** multipart/form-data — `file: UploadFile`, `session_id`, `language`, `model`
**Response 200:** `TranscriptionResponse` — `transcription_id`, `text`, `detected_language`, `processing_time_seconds`, `cost`, `model`
**Side effects:** sends to RunPod Whisper, inserts `transcriptions`, deducts tokens

## Internal — `/internal`

All internal endpoints require `X-Internal-Token` header. Blocked by ALB for external traffic.

### POST /internal/notify-user/{user_id}
**Auth:** Internal Token
**Request body:** notification payload
**Response 200:** emits Socket.IO event to user
**Used by:** worker process to push agent events to WebApp

### GET /internal/health
**Auth:** none
**Response 200:** `{"status": "ok"}`
